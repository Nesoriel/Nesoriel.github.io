## 项目背景

LWE（Lightweight Web Environment）是 Nesoriel 对静态 Web 与边缘托管边界的长期实验。它不试图把静态站点伪装成完整后端，而是把路由、构建产物、缓存策略与可观测信息整理成可审计的约定。

## 当前关注点

- 输出目录保持普通静态结构，便于迁移到不同平台。
- 将环境差异限制在构建期配置，而不是运行时判断。
- 为部署记录、站点地图和搜索索引保留稳定接口。

```ts
export interface StaticArtifact {
  path: string;
  contentType: string;
  immutable: boolean;
}
```

LWE 仍处在活跃设计阶段，仓库中的实现会优先服务 Nesoriel 官网本身。
